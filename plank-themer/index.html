Plank-Themer

Plank-Themer is a small program that allow you to change Plank Dock themes without any additional consumption of resources. Now change your theme easily and fast. Cheers!

Learn More

Screenshots

Super Easy to Install. Super Easy to Use.

The installation is a Ninja-Install that requires only one copy-paste on your Linux terminal.

Of course, you need to have Plank installed on your system.

Install Plank Cómo instalar Plank

Wait wait wait... BUT

BUT! Isn't that already in Settings > Elementary-tweaks? (In Elementary OS)

Well... yes... but NO.

First, it's true that you can change the themes from elementary-tweaks, but that only exists for Elementary OS Luna (or the old fashioned Ubuntu 12.04 with PPAs).

So, what if you are using something more modern, like Ubuntu 13.10, 14.04 or Fedora 20, you love Plank... and you can't live without changing your themes... then... ufff.... yeah. It sucks.

PLUS, the Elementary-Tweaks method is really unefficient, even within Elementary OS. It takes you lots of steps to do the stuff: 1) open settings 2) open e-tweaks 3a) Scroll down 3b) click on themes 4) select your theme (also scrolling). 5) in the mean time, this process eat your time, your patience and SPECIALLY your memory and resources.

This is a platform independent , on-dock approach.
That means that this solution is based on a dock entry. Just on the dock, on sight, at hand. Completely integrated with Plank. One click. No intermediate steps. No time spent. No processes in background. No resources involved. No Elementary OS or PPAs required. Runs in any version on GNU/Linux with Plank installed.

And you can change the theme houndried times with an impressive responsive result. Even after using your theme 10 mins... you want to change it back? one click. From all points of comparison, it is a better approach than elemematry tweaks. Nothing against it, of course (I LOVE IT!), but the motivation of building and using it is crystal clear. If you like to often change your Plank look, this is your stuff. :)

How it works

How to install: Just copy and paste!

Standard Install:

Open a Terminal in your Linux system. Then, paste this code (it is only one line): 

cd ~ && mkdir -p ~/.temp-plank-themer && cd ~/.temp-plank-themer && wget https://github.com/rhoconlinux/plank-themer/archive/master.zip && unzip master.zip && cd plank-themer-master/ && rm -fR ~/.config/plank/dock1/theme_index; rm -fR ~/.config/plank/dock1/themes-repo; cp -a theme_index/ ~/.config/plank/dock1 && cp -a themes-repo/ ~/.config/plank/dock1 && cd ~ && rm -R ~/.temp-plank-themer && sh ~/.config/plank/dock1/theme_index/plank-on-dock-themer.sh
Yes, you can close your terminal. :)

Install and ADD the themes:

Or you can install a version that also comes WITH A THEMES COLLECTION!: 

cd ~ && mkdir -p ~/.temp-plank-themer && cd ~/.temp-plank-themer && wget https://github.com/rhoconlinux/plank-themer/archive/master.zip && unzip master.zip && cd plank-themer-master/ && rm -fR ~/.config/plank/dock1/theme_index; rm -fR ~/.config/plank/dock1/themes-repo; cp -a theme_index/ ~/.config/plank/dock1 && cp -a themes-repo/ ~/.config/plank/dock1 && cd ~ && rm -R ~/.temp-plank-themer && sh ~/.config/plank/dock1/theme_index/plank-on-dock-themer.sh && cd ~/.config/plank/dock1/themes-repo/Themes/ && sudo cp -a . /usr/share/plank/themes

You can add the themes anytime by running this, after the standard install: 

 cd ~/.config/plank/dock1/themes-repo/Themes/ && sudo cp -a . /usr/share/plank/themes
